# Sensor_HC-SR04
Programação usual do  Sensor Ultrassônico HC-SR04 com Arduino Uno R3 para detecção de objetos a uma distância de 15 cm (Ajustável).  
:bell: O código é uma adaptação para a ausência de jumpers macho-femêa que seriam necessários para utilizar o sensor.  
:bell: Na montagem do sistema, deve ser unido os pinos Trig e Echo com um jumper macho-macho na protoboard.  
:bell: Desta forma, é enviado apenas um sinal para o Arduino, que com este código, separa-os e o sensor fica próprio para utilização.
